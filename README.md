Retos de la semana 1 de Data Academy de Platzi

### Instalación
Una vez que clones o bajes el repositorio. Dentro de la carpeta del proyecto hay que crear nuestro Virtual Environment

```
$ py -m venv venv
```

Esto creará tu entorno virtual de python y para activarlo ejecuta el siguient comando

Windows:
```
$ venv\Scripts\activate
```
Linux o Mac:
```
$ source venv/bin/activate
```

Esto te debe mostrar algo como:

```
(venv) D:\proyectos\python\datacademy\python_cardio [master ≡]
```

Dentro de aquí hay que declarar nuestro PYTHONPATH, ya que los scrpits realizados se encuentran modularizados, para esto corremos el siguiente comando:

Windows:
```
set PYTHONPATH=path\to\your\project\folder
```

Linux or Mac:
```
export PYTHONPATH="path\to\your\project\folder"
```
o:
```
export PYTHONPATH=$PWD
```
Una vez realizado estos pasos ya puedes entrar a cada reto y correr el script que desees observar
